A UX Research and Design Project


Silver Wallet is a class project from Northeastern University's Interaction Design: Mobile course. The project was created in Spring 2020 as one of the in class projects.


The purpose of the project is to conduct user research for the elderly population and come up with an mobile app that would fulfill their needs. The designed app must be user friendly towards elderly people.
